    PHP伪协议

    PHP支持的伪协议有以下几种

    allow_url_fopen 和allow_url_include这两个关键配置需要开启相应的服务，才能使用相关函数对伪协议的支持，配置相关在php.ini文件中

    php://协议

    php://是用来访问各个输入、输出流的。
    在php://下还有许多子协议

    php://input

    php://input代表可以访问请求的原始数据，简单来说POST请求的情况下，php://input可以获取到post的数据。
    php://output 是一个只写的数据流， 允许你以 print 和 echo 一样的方式 写入到输出缓冲区。
