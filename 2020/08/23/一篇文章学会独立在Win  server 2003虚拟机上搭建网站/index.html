一篇文章学会独立在win2003虚拟机上搭建网站

2020-08-23
-
工具人

本文学习：在Windows server 2003 虚拟机上的服务器搭建一个网站，非常详细，我都会了真的，有手就行。
Windows+iis+asp+access
环境：一台windows server 2003虚拟机。
准备工作什么也没有，开始搭建

开始测试搭建

打开虚拟机，
1.点击开始

2.点击“管理您的服务器”

3.点击“添加或删除角色”

4.出现预备步骤，直接点击下一步
等待配置

5.配置结束后，点击“自定义配置”，下一步

6.选择第三个“应用程序服务器”，下一步

7.两个都勾选上，下一步

8.直接下一步！
9.等待配置组件

10.配置成功。出现下图页面，点击完成。

11.点关闭页面，点击“开始菜单”——点击“管理工具”——点击第一项“Internet”。

12.打开后出现

13.右键单击“默认网站”——点击“打开”

14.选择全部文件，删除

15.右键单击——新建——文本文件，文件名为index.html

16.右键单击这个文件，选择编辑
编辑一段文字即可(内容随意)。

17.右键单击默认网站——选择权限

18.点击添加


19.选择高级

20.点击 “立即查找”——在下面找到“Everyone”

21.选好之后点击确定，再点击确定。(点两次)
选择完全控制，点击允许(因为是测试，所以可以设置成这样)，点击确定

22.右键单击默认网站，选择“属性”，点开后，找到“文档”打开，按照下图一步一步做，(默认内容文档全部删除)

23.点击添加，将自己编写的index.html写上，点击确定，再次确定。

24.回到桌面，点击打开浏览器，

25.输入127.0.0.1或者加上http(无所谓的)，就可以看到在index.html里写的内容了。

测试成功，下面开始搭建网站。
1.网站——新建——网站


2.点击下一步


3.描述可随意，点击下一步

4.直接点击下一步

补充知识：IP分配就是配置服务器的IP，别人可以通过本机(服务器)的IP地址，可以访问我们的网站。端口分配的话， 我在啃书的栏目中第一口。有对端口的理解，这里不再重复。这里不需要修改，想改动，后面也可以更改。
5.路径选择，浏览，选择C盘下的，Inetput文件夹下的wwwroot文件夹，点击确定。点击下一步

6.点击下一步，因为访问网站的用户，我们就让他可读就行。

7.点击完成

8.选择默认网站，点击停止，因为不能同时开两个网站。

9.右键点击自己创建的网站，点击启动。

10.自己在百度找一个asp源码，我选择这个。

我选择了第一个。

找到下载地址，随便挑个

11.将下载好的压缩包，拖进虚拟机中，

12.打开创建的网站

13.双击打开压缩包。打开文件，
全部选中文件剪切(或复制)到网站的源码文件夹

14.找找里面的index.asp,然后先关闭了这个文件夹
点击网站j4y，右键点击，选择属性，点击文档，全部删除内容文档。

15..添加index.asp点击确定就行了。
16.点击“主目录”，选择“执行权限”，选择纯脚本。

17.点击配置，

18.点击“选择”——启用父路径
19.选择“调试”——调试标志，两个全部勾选，点击确定。再次确定

20.点击Web服务扩展，点击Active Server Pages，点击允许。

21.点击j4y网站，右键点击，进行关闭和启动操作来达到重启的目的。
22.可以在浏览器中打开127.0.0.1打开网站。

点击添加就行了。
然后会进入了你的网站

23.那么这个时候如何让别人也能访问这个网站呢？？
这个时候就需要别人访问虚拟机的IP地址来访问网站了。
获取IP地址就行了。
打开cmd，输入命令ipconfig就能获取到IP地址了。

实验，我在主机上访问这个网站

到这里，一个简单的网站就搭建成功了。

总结：第二次搭建，速度快了很多。学到很多，自己搭建成功后很开心，有收获就开心。加油，继续努力。
