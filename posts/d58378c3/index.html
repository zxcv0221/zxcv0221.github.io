初学阶段对数据库、mysql等知识的较浅程度的理解

近段时间了解了数据库等知识的概念以及MySQL的初步学习，以下就是我在学习过程中的一些收获，跟大家分享一下。

数据库（DateBase,简称DS）

数据是数据库中存储的基本对象。数据的种类有很多种，文本（Text）、图形(graph)、图像（lmage）、音频（Audio）、视频（Video）、各种档案记录等等。这都是属于数据的部分。

  1. 数据库，概念：数据库顾名思义就是存放数据的仓库。数据库的存放空间很大，可以存放上亿条数据，但是数据库不是随意的将数据储存进仓库的，它是有一定规则的，因为这样保证了查询数据时的效率，否则查询效率会非常低。

    数据表（DateTable）

    数据表是一个临时保存数据的网格虚拟表，它不需要代码就可以简单的绑定数据库。
    当访问DateTable对象时，注意一点它们是区分大小写的。如果一个DateTable被命名时的名字大小写不同，那么它们就不是同一个数据表。
    数据表使用前需要自己创建一个。

    字段（field）

    在数据库中，大多数时，表的“列“称为”字段“，每个字段包含某一专题的信息。
    有时候，字段也不代表数据表中的列。

    MySQL

    MySQL是一种关系型数据库管理系统，关系数据库将数据保存在不同的表中，而不是将所有数据放在一个大仓库内，这样就增加了速度并提高了灵活性。该系统使用的语言是SQL语言。
    （SQL语句结束，要在语句末尾加上一个分号“ ； ”来结尾。）

    2.打开指定的库

    use 库名；（输入命令时，单词之间要加空格）

    3.查看当前库的所有表

    show tables;下图为具体示例。（注意：查看数据表前，要先打开你想查看的数据库，否则就会出现查询不到信息现象。）

    4.查询其他数据库的所有表

    show tables from 库名；

    5.查看表结构

    desc 表名；

6.查看表里的数据

select * from 表名；
输入此语句即可查看该表里的数据。

7.增添/创建一个数据库

create database test1；（库名）
这样就可以创建一个库名叫做test1的一个数据库，

8.创建表

可一次性创建多个表：
create table 表名（列名，列类型）；
也可以单独创建一个表：
create table 表名 （
列名 列类型，
列名 列类型
………

)；
注：创建一个表时，列类型后不用加“逗号”，创建多个表时，最后一个列表后不用加“逗号”。
